Programmes required to install in order to have the working prototype. 

To run the game you will need to: 

Download python 2.7: https://www.python.org/downloads/ 
Download Pygame for python 2.7: http://www.pygame.org/download.shtml 
Download Pandas: https://pypi.python.org/pypi/pandas 
Download Tkinter: https://pypi.python.org/pypi/EasyTkinter/1.1.0 
Download PySerial: https://pypi.python.org/pypi/pyserial/2.7 

To compile and run in terminal:

python main.py
